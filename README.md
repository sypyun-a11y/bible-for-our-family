# 말씀 — 개역한글 성경 앱

오늘의 말씀, 통독, 검색, 메모를 지원하는 한국어 성경 웹앱입니다.

## 기능

- **오늘의 말씀**: 날짜 기반 결정론적 추천 + 랜덤 모드
- **전체 읽기**: 66권 장 단위 네비게이션, 즐겨찾기, 절별 메모
- **검색**: 키워드 검색 (구약/신약 범위) + 책·장·절 직접 이동 (예: `요한복음 3:16`)
- **통독 계획**: 1년/반년/100일/90일 프리셋 + 기간·분량 커스텀
- **보관함**: 즐겨찾기 + 메모 모아보기

데이터는 모두 브라우저 `localStorage`에 저장됩니다 (로그인 없음).

## 데이터

개역한글 (KRV) 1961 — 공개 도메인.
출처: [laisiangtho/bible](https://github.com/laisiangtho/bible/blob/master/json/88.json)

성경 데이터를 다시 빌드하려면:

```bash
npm run data:build
```

`scripts/krv-raw.json`이 없으면 자동으로 다운로드합니다.

## 개발

```bash
npm install
npm run dev
```
